# Installation Review Guidance

## Rule

Contracts should clearly assign responsibility for installation dependencies, site readiness, utilities, permits, commissioning, and acceptance testing.

## Risk Signals

- Equipment delivery is scheduled before site readiness is confirmed.
- Civil works, utilities, permits, or shielding requirements are outside the vendor scope but not tracked as approval conditions.
- Commissioning responsibility is ambiguous.
- Acceptance criteria are missing or subjective.

## Follow-Up Questions

- Who owns each installation dependency?
- What proof is required before shipment or delivery?
- What happens if site readiness delays installation?
