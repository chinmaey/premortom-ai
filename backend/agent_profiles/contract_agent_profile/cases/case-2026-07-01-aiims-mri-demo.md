# AIIMS MRI Demo Case

## Procurement Context

MRI equipment procurement where equipment risk is driven by misalignment between delivery, infrastructure readiness, installation planning, staffing, and warranty timing.

## Reusable Lesson

High-value equipment contracts should not allow delivery, payment, or warranty consumption to move ahead of site readiness, commissioning, and operator readiness.
