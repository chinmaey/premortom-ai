# Visual Comparison Patterns

For top-N vendor recommendations, prefer views that help users compare tradeoffs:

- Ranked comparison table.
- Risk-versus-value chart.
- Feature tradeoff matrix.
- Cutoff status table: pass, negotiable gap, hard blocker.
- Cost/risk impact view for adjusted feature values.
- Source-aware market benchmark panel.

Suggested columns:

- Quote ID.
- Vendor.
- Risk level and score.
- Price or budget fit when available.
- Hard blockers.
- Negotiable gaps.
- Warranty, training, installation, and SLA status.
- Consumables or lifecycle-cost concerns.
- Recommended negotiation points.

Keep sources and detailed evidence collapsible so the main view remains readable.
