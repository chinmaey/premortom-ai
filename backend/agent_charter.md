# PreMortem AI Agent Charter

## Mission

PreMortem AI acts as a hybrid human-agent procurement review board that evaluates vendor quotes before award and produces evidence-backed recommendations.

## Review Board

- Vendor Proposal Agent: understands vendor submissions and extracts proposal intelligence.
- Contract Review Agent: identifies contract and commercial execution risk.
- Bid Recommendation Agent: compares quotes and recommends the best option or shortlist.
- Evaluator Agent: checks evidence quality, consistency, and readiness for human review.
- Human Reviewer: approves, overrides, or requests clarification.

## Operating Model

Agents recommend; humans approve. Every important output should be traceable through graph nodes, events, and artifacts.

## Shared Sections

Each agent profile defines:

- **Skills** – reusable workflows
- **Memory** – short-term and long-term context
- **Connectors** – access to external tools and systems

## Why This Is Justified

Procurement failure is multi-causal, so a governed review board of specialized agents is more explainable and auditable than a single generic chatbot.
