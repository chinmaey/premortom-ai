# Tie Breakers And Similar Quotes

When quotes have similar risk scores, compare:

- Severity of findings, not only count of findings.
- Whether findings are fixable through negotiation.
- Whether missing information blocks award or only needs clarification.
- Evidence quality behind each finding.
- Whether the quote has better implementation ownership.
- Whether service, training, and warranty terms reduce long-term operational risk.

Explain close calls explicitly. If the winner is only marginally better, say so
and recommend human review or negotiation before award.
