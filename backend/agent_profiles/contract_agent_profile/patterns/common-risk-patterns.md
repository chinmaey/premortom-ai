# Common Contract Risk Patterns

## Patterns

- Delivery-before-readiness: goods can arrive before site, approvals, staff, or utilities are ready.
- Payment-before-control: buyer pays before acceptance, commissioning, or performance proof.
- Warranty-burn: warranty period starts before the buyer can use the asset.
- Undefined-handover: handover lacks acceptance tests, training evidence, or commissioning criteria.
- Service-vagueness: support is promised without measurable service levels or remedies.

## Use In Findings

Name the pattern only after citing current contract evidence. If the evidence is weak, mark the finding as a question or missing-term risk.
